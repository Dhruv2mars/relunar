# @relunar/web

Marketing site and documentation for [Relunar](https://relunar.com).

## Develop

```sh
bun run dev --filter=@relunar/web
```

## Deploy on Vercel

- Root Directory: `apps/web`
- Install Command: `bun install`
- Build Command: `cd ../.. && bun run build --filter=@relunar/web`

Add logo assets to `public/brand/`:

- `relunar-blackfill.png`
- `relunar-whitefill.png`
